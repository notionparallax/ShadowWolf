Styleguide

So that all our project sheets look the same and read as if they are written with one voice; here are some guidelines.

General

As a base we use the Economist style guide. Variations from that are detailed below.

Measurements

All measurements are to be in SI units or SI derivatives.

Areas are in m²

Never use M², M2, m2, sqm, sq.m etc.
